# HR Attrition Analysis Dashboard (Power BI)

## Project Overview

This project analyzes employee attrition patterns to identify key factors affecting workforce retention.

The dashboard was built using Power BI and focuses on visualizing attrition trends across departments, age groups, and tenure.

## Objectives

* Analyze employee attrition rate
* Identify high-risk employee segments
* Understand department-wise attrition
* Provide insights for HR decision making

## Tools Used

* Power BI
* Data Modeling
* DAX Measures

## Key Insights

* Employees with lower tenure showed higher attrition risk
* Certain departments experienced higher turnover
* Attrition varied across age groups and job roles
